镜像加速 git clone
下载快，push 安全

从镜像站下载（快），克隆完成后自动重置 remote 为官方地址， 后续 pull / push 走官方仓库（安全）。 零外部依赖，仅用 Python 标准库。

自动保护

克隆失败概率极低

速度监控 + 自动重试 + 兜底直连，三重保障。

📊

速度监控

持续低于 1 MB/s 达 3 分钟 → 自动中止、清残留、换镜像。

🔁

连接重试

每个镜像连接失败自动重试 3 次，仍失败则换下一个镜像。

🎯

兜底直连

全部镜像失败后，最后尝试官方仓库直连，确保克隆能完成。
